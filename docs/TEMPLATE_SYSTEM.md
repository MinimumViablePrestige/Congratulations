# Система шаблонов MVP 1.0

## Принцип

В MVP шаблон — это базовый визуальный стиль, а не уникальный макет под каждого отдельного получателя.

Один шаблон должен работать для множества похожих сценариев за счет:

1. текста;
2. фото получателя;
3. цветовых вариаций;
4. обложки;
5. деталей анимации.

## Шаблон 1 — Теплый Классический

### Для кого

1. Учитель.
2. Воспитатель.
3. Благодарственные и уважительные поздравления.

### Настроение

Спокойный, теплый, аккуратный, уважительный.

### Визуальный характер

1. Светлая основа.
2. Мягкие кремовые и теплые песочные оттенки.
3. Легкие бумажные или акварельные ассоциации.
4. Деликатный декор без перегруза.

### Возможная палитра

1. `#F7F1E8`
2. `#D9C3A5`
3. `#8F6B4F`
4. `#3F342B`

### Рекомендуемая роль

Шаблон по умолчанию для сценариев `учитель` и `воспитатель`.

## Шаблон 2 — Командный Современный

### Для кого

1. Коллега.
2. Команда.
3. Рабочие и деловые поздравления без сухости.

### Настроение

Собранный, современный, дружелюбный, уверенный.

### Визуальный характер

1. Чистая композиция.
2. Больше графики, меньше декоративной “открыточности”.
3. Современные карточки и аккуратная сетка.
4. Легкое ощущение digital-подарка.

### Возможная палитра

1. `#F3F7FA`
2. `#9FC2D1`
3. `#27566B`
4. `#1E2A32`

### Рекомендуемая роль

Шаблон по умолчанию для сценария `коллега`.

## Шаблон 3 — Праздничный Яркий

### Для кого

1. Универсальные радостные поводы.
2. Более эмоциональные поздравления.
3. Сценарии, где нужен выраженный “вау”-эффект.

### Настроение

Живой, радостный, заметный, праздничный.

### Визуальный характер

1. Яркая обложка.
2. Конфетти, сияние, мягкое движение.
3. Более контрастные акценты.
4. Эффект праздника сразу при открытии.

### Возможная палитра

1. `#FFF4D6`
2. `#FFB703`
3. `#FB8500`
4. `#C94C1A`

## Шаблон 4 — Нежный Личный

### Для кого

1. Более личные, душевные и мягкие поздравления.
2. Сценарии, где важна эмоциональная теплота.

### Настроение

Интимный, светлый, личный, трогательный.

### Визуальный характер

1. Мягкие градиенты.
2. Воздушность.
3. Спокойная типографика.
4. Упор на фото получателя и тексты поздравлений.

### Возможная палитра

1. `#FAF4F2`
2. `#E7C8C4`
3. `#B97C73`
4. `#5E4744`

## Общие требования ко всем шаблонам

1. Должны хорошо смотреться на мобильном.
2. Должны поддерживать фото получателя на обложке.
3. Не должны ломаться от длинных поздравлений.
4. Должны иметь мягкую, но быструю анимацию открытия.
5. Должны сохранять ощущение “подарка”, а не просто страницы с текстом.

## Вариативность в рамках MVP

Чтобы не раздувать первую версию, вариативность делаем ограниченной:

1. 4 базовых шаблона.
2. По 2-3 цветовые или декоративные вариации внутри каждого.
3. Автовыбор шаблона по сценарию с возможностью ручного переключения.
