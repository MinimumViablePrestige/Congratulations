# Настройка GitHub remote

## Текущее состояние

Локальный git-репозиторий уже инициализирован.

Удаленный репозиторий GitHub уже подключен как `origin`:

`https://github.com/barsev174-star/capsule_of_congratulations.git`

Попытка `push` из текущей среды пока не завершилась успешно и требует дополнительной проверки доступа или сетевых ограничений.

## Что нужно от пользователя

1. Если репозиторий пустой и доступ есть, помочь только если потребуется ручная авторизация GitHub на этой машине.
2. Если `push` не пройдет и дальше, подтвердить удобный способ авторизации: HTTPS или SSH.

## Что сделаю после этого

1. Проверю настройки ветки `main`.
2. Повторю `push`.
3. Если упрется в авторизацию, точно скажу, какой шаг нужен с твоей стороны.

## Рекомендуемый вариант

Если GitHub уже авторизован на этой машине, проще всего будет использовать HTTPS URL репозитория.
