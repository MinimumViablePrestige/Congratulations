# MVP 1.0 — зафиксированный продуктовый фокус

## Цель MVP

Проверить, готовы ли пользователи платить за простой способ организовать групповое поздравление и превратить его в красивый цифровой подарок.

## Что продаем пользователю

Пользователь платит не за “ссылку”, а за сочетание ценности:

1. Простая организация поздравления.
2. Помощь участникам с текстом через AI.
3. Красивый финальный экран открытия открытки.
4. Ощущение готового эмоционального подарка.

## Целевые сценарии первого релиза

1. Поздравление `учителю`.
2. Поздравление `воспитателю`.
3. Поздравление `коллеге`.

## Состав MVP 1.0

1. Лендинг с понятным оффером.
2. Создание открытки без регистрации.
3. Ссылка для участников.
4. Простое добавление поздравления.
5. AI-помощник для написания поздравления.
6. Панель организатора.
7. Предпросмотр финальной открытки.
8. Один тариф на публикацию.
9. Финальная ссылка после оплаты.
10. Базовая аналитика.

## Что важно для оплаты

Критически важны:

1. Красивый внешний вид финальной открытки.
2. Простота создания и рассылки.
3. Высокая конверсия участника в добавление поздравления.
4. Ощутимый “вау”-эффект при открытии.

## Шаблоны MVP

В первом релизе используются 4 базовых шаблона:

1. `Теплый Классический`
   - для учителя;
   - для воспитателя;
   - для уважительных и благодарственных сценариев.
2. `Командный Современный`
   - для коллег;
   - для рабочих и командных поздравлений.
3. `Праздничный Яркий`
   - для более эмоциональных и радостных поводов.
4. `Нежный Личный`
   - для душевного, личного и более мягкого визуального тона.

## Правило по шаблонам

В MVP шаблон — это не отдельный дизайн “под каждого учителя”, а базовый визуальный стиль с возможными легкими вариациями:

1. цвет;
2. декоративные элементы;
3. обложка;
4. подстройка под выбранный сценарий по умолчанию.

## Что отложено после MVP

1. Несколько тарифов.
2. Широкая SEO-сетка.
3. Фото в каждом поздравлении.
4. Видео- и голосовые поздравления.
5. Интеграции с Telegram и VK.
6. Корпоративные кабинеты.
7. Автоматические напоминания.

## Что можно быстро добавить после первого подтверждения спроса

1. PDF-экспорт.
2. Реакции на поздравления.
3. Аватары или инициалы авторов.
4. Итоговый блок “от всех” на основе собранных поздравлений.
